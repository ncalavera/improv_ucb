## General Tips for Image Generators

### For DALL-E / ChatGPT:
- Copy the prompt exactly as written
- Request "diagram" or "educational chart" style
- If text is unclear, you may need to regenerate or edit manually

### For Midjourney:
- Add `--style raw` for cleaner diagrams
- Add `--ar 16:9` for landscape orientation (good for flowcharts)
- Add `--ar 4:5` for portrait orientation (good for vertical flows)

### For Stable Diffusion:
- Use models optimized for diagrams/infographics
- Consider using ControlNet for precise layouts
- May need to add negative prompts: "photo, realistic, 3d, decorative"

### Important Notes:
- **All text must be in Russian (Cyrillic)**
- **No metaphors or decorative elements**
- **Focus on clarity and educational value**
- **Professional, clean design**
- If the generator struggles with Cyrillic text, you may need to:
  - Generate the diagram structure first
  - Add Russian text manually using image editing software
  - Or use a different generator that handles Cyrillic better
