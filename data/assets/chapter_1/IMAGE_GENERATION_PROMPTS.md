# Chapter 1 Image Generation Prompts

This document contains all prompts for generating educational diagrams for Chapter 1. These are logical schemas without metaphors, designed for maximum clarity using a simple stick figure (xkcd-like) style.

## Image Generation Instructions

When quota resets, generate these images using the prompts below and save them with the specified filenames in `assets/chapter_1/`.

---

## 01. "Yes, And" Building Blocks
**Filename:** `01_yes_and_blocks.png`
**Status:** ⏳ PENDING

**Prompt:**
```
Create a simple educational diagram using stick figure style (xkcd-like). Two stick figures building a structure with large blocks. Bottom block labeled "ДА" (YES) with text "Согласие с реальностью" (Agreeing with reality). Top block being placed by a stick figure labeled "И" (AND) with text "Добавление новой информации" (Adding new information). Together they form a solid structure. Simple black line drawings on white background. Clean sans-serif Cyrillic typography. Minimal style. 4:3 aspect ratio.
```

## 02. Base Reality Components (Who, What, Where)
**Filename:** `02_base_reality_components.png`
**Status:** ⏳ PENDING

**Prompt:**
```
Create a simple educational diagram using stick figure style (xkcd-like). A scene showing two stick figures in a kitchen (simple lines for table/stove). Arrows pointing to elements:
1. Pointing to the figures: "КТО" (Who) - Relationship/Character.
2. Pointing to the action (cooking/eating): "ЧТО" (What) - Activity.
3. Pointing to the room/background: "ГДЕ" (Where) - Location.
Title or central label: "БАЗОВАЯ РЕАЛЬНОСТЬ" (Base Reality). Simple black line drawings on white background. Clean sans-serif Cyrillic typography. Minimal style. 4:3 aspect ratio.
```

## 03. Initiation Flow
**Filename:** `03_initiation_flow.png`
**Status:** ⏳ PENDING

**Prompt:**
```
Create a linear process flow diagram using stick figure style (xkcd-like). Three steps connected by arrows.
Step 1: "ПРЕДЛОЖЕНИЕ" (Suggestion) - Stick figure audience member shouting a word (e.g., in a speech bubble "СЛОВО!").
Step 2: "АССОЦИАЦИЯ" (Association) - Stick figure improviser thinking (thought bubble with a lightbulb or connection).
Step 3: "ИНИЦИАЦИЯ" (Initiation) - Stick figure speaking the first line (speech bubble).
Simple black line drawings on white background. Clean sans-serif Cyrillic typography. Minimal style. 4:3 aspect ratio.
```

## 04. Character of the Space (Agreement)
**Filename:** `04_space_agreement.png`
**Status:** ⏳ PENDING

**Prompt:**
```
Create a split-panel comparison using stick figure style (xkcd-like).
Left panel: Stick figure 1 shivering, arms wrapped around self. Label: "Игрок 1: Дрожит" (Player 1: Shivers).
Right panel: Stick figure 2 putting on a coat/scarf. Label: "Игрок 2: Одевается" (Player 2: Dresses).
Large text connecting them: "СОГЛАСИЕ" (Agreement).
Visual cues of cold (simple snowflakes). Simple black line drawings on white background. Clean sans-serif Cyrillic typography. Minimal style. 4:3 aspect ratio.
```

## 05. Object Work: Show vs Tell
**Filename:** `05_object_work_show_tell.png`
**Status:** ⏳ PENDING

**Prompt:**
```
Create a comparison diagram using stick figure style (xkcd-like).
Left side (Bad): Stick figure standing still with speech bubble "Я мою посуду" (I am washing dishes). Large "X" mark. Label: "РАССКАЗЫВАТЬ" (Telling).
```
